# Hệ thống quản lý phòng trọ
quản lý phòng, hợp đồng, hoá đơn, thiết kế, thống kê và báo cáo
QUẢN LÝ PHÒNG TRỌ
Giới thiệu

Chương trình Quản lý phòng trọ giúp quản lý phòng, người thuê và hóa đơn bằng Python, dữ liệu được lưu trữ bằng file JSON.
Đề tài phục vụ học tập và rèn luyện kỹ năng lập trình.

Chức năng

Thêm, sửa, xóa, xem danh sách phòng

Thêm, xóa, xem người thuê

Quản lý hóa đơn

Thống kê số phòng trống, tổng tiền hóa đơn

Công nghệ

Python 3

JSON

Visual Studio Code

GitHub

Cấu trúc chính
data/           : lưu dữ liệu JSON
room.py         : quản lý phòng
tenant.py       : quản lý người thuê
invoice.py      : quản lý hóa đơn
report.py       : thống kê
menu.py         : menu chương trình
main.py         : chạy chương trình

Cách chạy
python main.py
